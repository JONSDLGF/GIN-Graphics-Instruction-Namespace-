# GIN

**One API, See Everything**

## Project

* **Name:** GIN
* **Author:** Jon

## Goal

GIN is a lightweight graphics abstraction layer.

Applications interact with GIN, and GIN renders through different backends:

* Canvas
* WebGL
* OpenGL
* Vulkan
* Other supported renderers

The application should not need to know which graphics backend is being used.
